Add a new feature tab to my existing Voice Persona AI app called “Travel Talk” (Translate), without changing or breaking any of the current tabs, features, or logic.

IMPORTANT:
- Do NOT modify existing features: Voice Studio, Creator Mode, History, or Home
- This must be implemented as a completely separate module
- Maintain the same premium dark UI style and design consistency

-----------------------------------
GOAL
-----------------------------------
Enable real-time, two-way communication between people who speak different languages.

The app should:
1. Let user speak in their own language
2. Convert speech to text
3. Translate it into a selected target language
4. Speak the translated text aloud using system/device voice
5. Listen to the other person’s reply
6. Translate that reply back into the user’s language

-----------------------------------

-----------------------------------

- Name: Travel Talk (or Translate)

Updated navigation:
- Home
- Studio
- Create
- Travel Talk
- History

-----------------------------------

Screen Title:
Travel Talk

MODE SELECTION
-----------------------------------
Add toggle or tabs:
- Speak Out
- Listen Back

SPEAK OUT FLOW
-----------------------------------
Purpose: User speaks → app translates → app speaks

UI:
- Large microphone button
- Status text: “Listening…”
- After recording, display:
  - Original Text
  - Translated Text
- Button: “Speak Translation”

Behavior:
- Use speech-to-text to capture user voice
- Translate text into target language
- Use system/device TTS (NOT cloned voice) to speak output

-----------------------------------
LISTEN BACK FLOW
-----------------------------------
Purpose: Other person speaks → app translates back

UI:
- Large microphone button
- Status text: “Listening to reply…”
- Display:
  - Heard Text
  - Translation (in user’s language)
- Button: “Read Aloud”

Behavior:
- Capture incoming speech
- Convert to text
- Translate into user’s language
- Optionally speak using system voice
