NEW TAB NAME:

GOAL:
Capture live microphone audio, stream it for real-time transcription using ElevenLabs, automatically detect the spoken language when possible, and display instant captions below. The app should also translate the transcript into English and show that English caption as the main output.

FEATURE BEHAVIOR:
1. Add a new tab button labeled "Live Captions".
2. On this tab, create:
   - Start Listening button
   - Stop Listening button
   - Live status indicator: Idle / Listening / Processing / Error
   - Small settings section
   - Caption panel with:
     - Detected Language
     - Original Transcript
     - English Translation
3. The English Translation should be visually emphasized as the main caption output.
4. Update captions continuously in near real time as speech is received.
5. Show partial transcript updates smoothly without page refresh.

SETTINGS / KEYS:
Support 2 modes inside this tab:

MODE A: User Key Mode
- Add optional inputs:
  - ElevenLabs API Key
- Add a button:
  - Save Key
- Store the key only locally in browser localStorage for that device/session.
- Do NOT hardcode the key in frontend code.
- If a key is present, enable live transcription.

MODE B: Demo / Fallback Mode
- If no ElevenLabs key is provided, do not break the feature.
- Show Demo Mode with:
  - sample multilingual text choices
  - manual text input
  - source language selector
  - translate to English button
- Demo mode should let users test the UI even without a key.

AUDIO CAPTURE:

- Prefer a streaming-friendly approach.
